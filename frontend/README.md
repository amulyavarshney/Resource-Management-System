## Introduction

This is the frontend of the VTS Timesheets Website. It is built using Next.js, NextAuth.js, TypeScript, and Tailwind CSS.

## Getting Started

To install the project, follow these steps:
- Clone the repository: `git clone https://ZEISSgroup-VIS@dev.azure.com/ZEISSgroup-VIS/VTS-ManagementTools/_git/VTS-Timesheets_Frontend`
- Install the dependencies: `npm install`
- Configure the environment variables: Create a `.env` file and add the necessary environment variables.
- Start the development server:
```bash
npm run dev
```
- Open your browser and navigate to `http://localhost:3000`

## Build and Test

- To build the project for production, run the following command:
```bash
npm run build
```
This will create a production-ready build of the project in the `.next` directory.

- To lint the code, run the following command:
```bash
npm run lint
```
This will run the ESLint linter on the project.
